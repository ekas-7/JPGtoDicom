# JPG to DICOM

## Running the Script

1. Create a virtual environment:

    ```bash
    conda create --name venv
    ```

2. Activate the virtual environment:

    ```bash
    conda activate venv
    ```

3. Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

4. Run the application:

    ```bash
    python app.py
    ```
